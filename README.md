Bowtie Editor
===

This repo holds our editor, which is a standalone component that edits content.

## What this repo will **not** include

* File navigation
* Networking


## What this repo includes

* WSYIWYG Editor
* CodeMirror powered code editor
* Packaged react component

### Props

* Content
* Front Matter

### State

* content: string
* hasChangedSinceLastSave: bool
* front_matter: object
